=====================================================
Настройка плагина для сервиса "СМС Биллинг Украина"
-----------------------------------------------------
Назначение: Скрытие текста помещенного между специальными тегами
Проверено на 3.1.1 версии
=====================================================

Подключение:

1. Скопируйте в папку wp-content/plugins/ на хостинге попку из архива.
2. Залогинтесь в админ панель сайта. Зайдите в раздел "Плагины" и активируйте плагин "SmsBill.com.ua Service"
3. В разеделе "Параметры" найдите "SmsBill Сервис" и зайдите внутрь. Здесь необходимо узаказь:
- ID услуги, поключенной в смс-биллинге;
- кодировку сайта;
- опционально можно указать путь к своему CSS файлу.
Нажмите кнопку "Сохранить".

Использование:
При созаднии материала на сатйе, поместите скрытый текст между специальными тегами, ваот так:

[smsbill_getpass] текст, который будет скрыт [/smsbill_getpass]

Полсе этого скрытый текст будет заменен на сайте на платежную форму с инструкциями к оплате.
